### 3. Connect Local Repository to GitHub
Replace `YOUR_REPO_NAME` with the repository name you created:

```bash
git remote add origin https://github.com/Ash456rt/YOUR_REPO_NAME.git
git branch -M main
git push -u origin main
```

### 4. Deploy to Vercel (Live Website)
1. Go to [vercel.com](https://vercel.com)
2. Sign in with your GitHub account
3. Click "Import Project"
4. Select your portfolio repository
5. Click "Deploy"
6. Your live portfolio will be at: `https://your-repo-name.vercel.app`
